# 🎓 Student Result Prediction System

A Machine Learning application that predicts student average scores using Linear Regression and provides predictions through a Streamlit web application.

---

## 🚀 Features

- 📊 Student performance prediction
- 🤖 Machine Learning model (Linear Regression)
- 📈 Model evaluation using MAE and R² Score
- 🌐 Interactive Streamlit web application
- 💾 Saved trained model (`student_model.pkl`)

---

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit
- Git & GitHub
